Boletim de Segurança da Microsoft MS15-001 - Importante
=======================================================

Vulnerabilidade no cache de compatibilidade de aplicativos do Windows pode permitir a elevação de privilégio (3023266)
----------------------------------------------------------------------------------------------------------------------

Publicado em: 13.01.15

**Versão:** 1.0

Sinopse
-------


Esta atualização de segurança resolve uma vulnerabilidade relatada de forma privada no Microsoft Windows. A vulnerabilidade pode permitir elevação de privilégio se um invasor fizer logon em um sistema e executar um aplicativo especialmente criado. Um invasor autenticado que explorou essa vulnerabilidade com êxito pode ignorar as verificações de permissão existentes que foram realizadas durante a modificação do armazenamento em cache no componente de Compatibilidade de Aplicativos do Microsoft Windows e executar o código arbitrário com privilégios elevados.

Esta atualização de segurança é classificada como Importante para todas as edições suportadas do Windows 7, Windows Server 2008 R2, Windows 8, Windows Server 2012, Windows RT, Windows 8.1, Windows Server 2012 R2 e Windows RT 8.1. Para obter mais informações, consulte a seção **Softwares afetados**.

A atualização de segurança trata a vulnerabilidade alterando como a infraestrutura de compatibilidade de aplicativos do Windows processa o uso de token de representação. Para obter mais informações sobre a vulnerabilidade, consulte a seção **Informações sobre vulnerabilidade**.


Para obter mais informações sobre essa atualização, consulte o [artigo 3023266 da Base de Dados de Conhecimento da Microsoft](https://support.microsoft.com/kb/3023266/pt-br).

Informações sobre a vulnerabilidade
-----------------------------------


Elevação de vulnerabilidade de privilégio da infraestrutura de compatibilidade de aplicativos da Microsoft - CVE-2015-0002
--------------------------------------------------------------------------------------------------------------------------

Existe uma elevação de vulnerabilidade de privilégio na forma da infraestrutura de compatibilidade de aplicativos do Microsoft Windows ([AppCompat](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx)) verificar a autorização do token de representação do chamador incorretamente. Um invasor pode tentar explorar isso para executar um aplicativo com privilégios. A atualização trata a vulnerabilidade, implementando a verificação de autorização adequada do uso de token de representação.

Essa vulnerabilidade foi divulgada de forma pública. Ela recebeu o número [CVE-2015-0002](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2015-0002) da lista Common Vulnerability and Exposure. Quando este boletim de segurança foi emitido, a Microsoft não estava ciente de ataques que tentaram explorar essa vulnerabilidade.

### Fatores atenuantes

A Microsoft não identificou quaisquer [fatores atenuantes](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx) para essa vulnerabilidade.

### Soluções alternativas

A Microsoft não identificou nenhuma [solução alternativa](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx) para esta vulnerabilidade.
