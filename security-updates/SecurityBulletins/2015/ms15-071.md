Boletim de Segurança da Microsoft MS15-071 - Importante
=======================================================

Vulnerabilidade no Netlogon pode permitir a elevação de privilégio (3068457)
----------------------------------------------------------------------------

Publicado em: 14.07.2015

**Versão:** 1.0

Sinopse
-------

Esta atualização de segurança resolve uma vulnerabilidade no Microsoft Windows. A vulnerabilidade pode permitir a elevação de privilégio se um atacante com acesso a um controlador de domínio primário (PDC) em uma rede de destino executa um aplicativo especialmente criado para estabelecer um canal seguro com o PDC como um controlador de domínio de back-up (BDC).

Esta atualização de segurança é classificada como Importante para todas as edições suportadas do Windows Server 2003, Windows Server 2003 R2, Windows Server 2008, Windows Server 2008 R2, Windows Server 2012 e Windows Server 2012 R2. Para obter mais informações, consulte a seção **SoftwareAfetado**.

A atualização elimina a vulnerabilidade modificando a maneira como o Netlogon lida com o estabelecimento de canais seguros. Para obter mais informações sobre a vulnerabilidade, consulte a seção **Informações sobre** **vulnerabilidade**.

Para obter mais informações sobre essa atualização, consulte o [artigo 3068457 da Base de Dados de Conhecimento da Microsoft](https://support.microsoft.com/pt-br/kb/3068457).

Software Afetado
----------------

As seguinte versões ou edições de software foram afetadas. As versões ou edições que não estão listadas já passaram de seu ciclo de vida do suporte ou não foram afetadas. Para determinar o ciclo de vida do suporte para sua versão ou edição de software, visite o site [Ciclo de vida do suporte da Microsoft](http://support.microsoft.com/default.aspx?scid=fh;%5Bln%5D;lifecycle).

Informações sobre a vulnerabilidade
-----------------------------------

Vulnerabilidade de elevação de privilégio no Netlogon - CVE-2015-2374
---------------------------------------------------------------------

Existe uma vulnerabilidade de elevação de privilégio no Netlogon que é causada quando o serviço estabelece inadequadamente um canal de comunicação seguro para um controlador de domínio primário (PDC). Para explorar com êxito esta vulnerabilidade, um atacante precisaria primeiro ter acesso a um PDC em uma rede de destino. Um atacante pode então executar um aplicativo especialmente criado que pode estabelecer um canal seguro para o PDC como um controlador de domínio de back-up (BDC) e pode ser capaz de divulgar credenciais. Os servidores configurados como controladores de domínio estão em risco com essa vulnerabilidade. A atualização resolve a vulnerabilidade modificando a maneira como o Netlogon lida com o estabelecimento de canais seguros.

A Microsoft recebeu informações sobre esta vulnerabilidade por meio da divulgação coordenada de vulnerabilidades. Quando esse boletim de segurança foi originalmente emitido, a Microsoft ainda não tinha recebido informações que indicassem que a vulnerabilidade tinha sido usada de forma pública para atacar clientes.

### Fatores atenuantes

A Microsoft não identificou quaisquer [fatores atenuantes](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx) para essa vulnerabilidade.

### Soluções alternativas

A Microsoft não identificou nenhuma [solução alternativa](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx) para esta vulnerabilidade.
