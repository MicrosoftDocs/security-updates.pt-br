Boletim de Segurança da Microsoft MS15-003 - Importante
=======================================================

Vulnerabilidade no Serviço de Perfil de Usuário do Windows pode permitir a elevação de privilégio (3021674)
-----------------------------------------------------------------------------------------------------------

Publicado em: 13.01.15

**Versão:** 1.0

Sinopse
-------

<span id="sectionToggle0"></span>
Essa atualização de segurança elimina uma vulnerabilidade divulgada publicamente no Microsoft Windows. A vulnerabilidade pode permitir elevação de privilégio se um invasor faz logon no sistema e executar um aplicativo especialmente criado. Um invasor local, que explorou com êxito esta vulnerabilidade, pode executar o código arbitrário em um sistema de destino com privilégios elevados. O invasor precisa ter credenciais de logon válidas e poder fazer logon localmente para explorar essa vulnerabilidade.

Essa atualização de segurança foi classificada como Importante para todas as versões com suporte do Microsoft Windows. Para obter mais informações, consulte a seção **Softwares afetados**.

<span id="KBArticle"></span>
A atualização de segurança trata a vulnerabilidade corrigindo como o Serviço de Perfil de Usuário do Windows valida privilégios do usuário para carregar hives do registro. Para obter mais informações sobre a vulnerabilidade, consulte a seção **Informações sobre vulnerabilidade**.

Para obter mais informações sobre essa atualização, consulte o [artigo 3012674 da Base de Dados de Conhecimento da Microsoft](https://support.microsoft.com/kb/3012674/pt-br).

Informações sobre a vulnerabilidade
-----------------------------------

<span id="sectionToggle3"></span>
Elevação de vulnerabilidade de privilégio do Serviço de Perfil de Usuário da Microsoft – CVE-2015-0004
------------------------------------------------------------------------------------------------------

A vulnerabilidade que permite uma elevação de privilégio está no modo como o Serviço de Perfil de Usuário do Windows valida o privilégio de usuário. Um invasor autenticado que explora a vulnerabilidade com êxito pode aproveitar o Serviço de Perfil de Usuário (ProfSvc) do Windows para carregar hives do registro associados a outras contas de usuários e executar potencialmente programas com permissões elevadas. A atualização de segurança trata a vulnerabilidade corrigindo como o Serviço de Perfil de Usuário do Windows valida privilégios do usuário para carregar hives do registro.

Essa vulnerabilidade foi divulgada de forma pública. Foi atribuído um número de Vulnerabilidade e Exposição Comum [CVE-2015-0004](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2015-0004). Quando esse boletim de segurança foi emitido, a Microsoft ainda não tinha recebido informações que indicassem que a vulnerabilidade tinha sido usada de forma pública para atacar consumidores.

### Fatores atenuantes

A Microsoft não identificou quaisquer [fatores atenuantes](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx) para essa vulnerabilidade.

### Soluções alternativas

A Microsoft não identificou nenhuma [solução alternativa](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx) para essa vulnerabilidade.** **
