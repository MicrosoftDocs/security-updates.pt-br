Microsoft Security Bulletin MS15-085 - Importante
=================================================

Vulnerabilidade no Gerenciador de montagem pode permitir a elevação de privilégio (3082487)
-------------------------------------------------------------------------------------------

Publicado em: 11.08.15

**Versão:** 1.0

Sinopse
-------

Esta atualização de segurança resolve uma vulnerabilidade no Microsoft Windows. A vulnerabilidade pode permitir a elevação de privilégio se um atacante inserir um dispositivo USB mal-intencionado no sistema de destino. Um atacante poderia então escrever um binário mal-intencionado para o disco e executá-lo.

Essa atualização de segurança foi classificada como Importante para todas as versões com suporte do Microsoft Windows. Para obter mais informações, consulte a seção **Software Afetado**.

A atualização de segurança elimina a vulnerabilidade removendo o código vulnerável do componente Gerenciador de montagem. Para obter mais informações sobre a vulnerabilidade, consulte a seção **Informações sobre vulnerabilidade**.

Para obter mais informações sobre essa atualização, consulte o [artigo 3071756 da Base de Dados de Conhecimento da Microsoft](https://support.microsoft.com/pt-br/kb/3071756).

Software Afetado
----------------

As seguinte versões ou edições de software foram afetadas. As versões ou edições que não estão listadas já passaram de seu ciclo de vida do suporte ou não foram afetadas. Para determinar o ciclo de vida do suporte para sua versão ou edição de software, visite o site [Ciclo de vida do suporte da Microsoft](http://support.microsoft.com/default.aspx?scid=fh;%5Bln%5D;lifecycle).

Informações sobre a vulnerabilidade
-----------------------------------

Vulnerabilidade de elevação de privilégio no Gerenciador de montagem - CVE-2015-1769
------------------------------------------------------------------------------------

Existe uma vulnerabilidade de elevação de privilégio quando o componente Gerenciador de montagem processa impropriamente os links simbólicos. Um atacante que explorar esta vulnerabilidade com êxito pode escrever um binário mal-intencionado para o disco e executá-lo.

Para explorar a vulnerabilidade, um atacante teria que inserir um dispositivo USB mal-intencionado em um sistema de destino. A atualização de segurança elimina essa vulnerabilidade removendo o código vulnerável do componente.

A Microsoft recebeu informações sobre esta vulnerabilidade por meio da divulgação coordenada de vulnerabilidades. Quando este boletim foi emitido, a Microsoft tinha motivos para acreditar que esta vulnerabilidade foi usada em ataques dirigidos contra clientes.

### Fatores atenuantes

A Microsoft não identificou quaisquer [fatores atenuantes](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx) para essa vulnerabilidade.

### Soluções alternativas

A Microsoft não identificou nenhuma [solução alternativa](https://technet.microsoft.com/pt-br/library/security/dn848375.aspx) para esta vulnerabilidade.
